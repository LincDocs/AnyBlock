Tow-Layer-Tree

和 Mindmaps 类别相似，也可以表示成树，区别在于深度只有两层

其中二层树又分为二层一叉树和二层多叉树

二层一叉树就是的第一层为标题，第二层为主要内容。标识的选择器有：

  • 分栏
  • 卡片
  • 标签栏
  • 时间线
