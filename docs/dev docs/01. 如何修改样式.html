如何修改样式

在哪个位置进行修改？

  1. (不推荐) 进入插件所在文件夹，修改其 styles.css
  2. (推荐) Obsidian设置 > 外观 > 添加CSS片段
  3. (推荐) 不修改原有的插件内部class，而是使用 addClass() 处理器添加新class，然后再根据自己的新class去修改外观

不会CSS？简易教程见：https://www.runoob.com/css/css-tutorial.html
